# PhoneBook-App
Web Application with React.js

Stack technologies :

    Node.js (express)
    React redux
    Firebase real time and GraphQl as database
    NPM (others package)
    Bootstrap 4.5 (View)
    
FEATURE :

Search contacts

Add,Edit and Delete Contacts

Included with pagination
